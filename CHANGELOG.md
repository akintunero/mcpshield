# Changelog

## 1.0.0 (2026-07-14)

### Initial Release

- **AI Security Analyst**: Slack-integrated AI agent with natural language interaction
- **MCP Server**: 11 standardized tools for cloud security operations
- **Plugin Architecture**: `CloudProvider` interface supporting multi-cloud extensibility
- **Declarative Rules Engine**: YAML-defined security rules with dynamic loading
- **Resource Graph**: Relationship mapping between cloud resources
- **Audit Trail**: Persistent log of every AI action for transparency
- **15 Security Rules**: Critical, High, Medium, and Low findings with full MITRE ATT&CK, CIS, and NIST mappings
- **LocalStack Integration**: Complete mock AWS environment
- **Terraform Remediation**: Auto-generated HCL for every finding
- **AWS CLI Remediation**: Auto-generated CLI commands
- **Human-in-the-Loop**: Approval workflow for all write operations
- **Executive Reports**: Professional security posture assessment
- **Multiple LLM Providers**: NVIDIA NIM, Gemini, Ollama, OpenAI-compatible
- **Web Dashboard**: Real-time security posture visualization
- **Workshop Curriculum**: 10 guided labs with facilitator guide
- **n8n Workflows**: Pre-built automation workflows
