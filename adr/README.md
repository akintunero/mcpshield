# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) documenting the key architectural decisions made in MCPShield.

## What is an ADR?

An Architecture Decision Record is a short document capturing an important architectural decision, its context, options considered, and the rationale for the chosen approach.
